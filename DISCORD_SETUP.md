# Discord Channel Setup Instructions

## Step 1: Create Dedicated Channel
1. In your Discord server, click the **+** next to a category
2. Select **Text Channel**
3. Name: `trading-heartbeats` (or whatever you prefer)
4. Click **Create**

## Step 2: Create Webhook (for automated posts)
1. Go to channel settings (gear icon)
2. Click **Integrations**
3. Click **Webhooks** → **New Webhook**
4. Name it: `PolyClaw Trader`
5. Copy the **Webhook URL**
6. Give it to me: `https://discord.com/api/webhooks/...`

## Step 3: Give Bot Permissions
1. Go to Server Settings → Roles
2. Find your bot's role
3. Make sure it has:
   - Send Messages
   - Embed Links
   - Attach Files
   - Read Message History
