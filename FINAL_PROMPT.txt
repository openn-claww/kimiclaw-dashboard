Problem:
- Trades are marked as "won" with exit_price=1.0
- But pnl_usd = $0.00 for all trades
- Expected: Win should show positive PnL

Trade data from pnl_trades.json:
```
BTC-5m YES @ $0.505 -> $1.0 | won | PnL: $+0.00
ETH-5m YES @ $0.51 -> $1.0 | won | PnL: $+0.00
```

Calculation in pnl_tracker.py close() method:
```python
def close(self, exit_price: float, exit_reason: str = ''):
    self.exit_price = exit_price
    self.exit_ts = datetime.now(timezone.utc).isoformat()
    self.exit_reason = exit_reason
    
    # PnL calculation
    if self.side == 'YES':
        gross = (exit_price - self.entry_price) * self.shares
    else:
        gross = (self.entry_price - exit_price) * self.shares
    
    self.fee = abs(gross) * 0.02  # 2% fee
    self.pnl_usd = gross - self.fee
    
    if self.pnl_usd > 0:
        self.status = 'won'
    elif self.pnl_usd < 0:
        self.status = 'lost'
    else:
        self.status = 'breakeven'
```

QUESTION: Why is PnL $0 when exit_price ($1.0) > entry_price ($0.505)?
Is the shares calculation wrong? Entry showing $0.505 but maybe shares=0?
